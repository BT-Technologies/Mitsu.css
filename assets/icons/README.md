# Mitsu UI Icons (32x32)

Sprite file: `assets/icons/mitsu-ui-32.svg`

Usage:

```html
<svg class="mitsu-icon mitsu-icon-32" aria-hidden="true">
  <use href="assets/icons/mitsu-ui-32.svg#mi-home"></use>
</svg>
```

Available sizes:

- `.mitsu-icon-32`
- `.mitsu-icon-24`
- `.mitsu-icon-16`

The icon set is stroke-based and uses `currentColor`, so icons inherit text color automatically.
