Layout Primitives

Use .mitsu-container, .mitsu-row, .mitsu-stack-*, and .mitsu-grid for composition, then combine them into complete page patterns.

Think in layers: container controls width, row/grid control structure, and stack controls vertical rhythm. Avoid custom one-off wrappers when a primitive already exists.

.mitsu-container .mitsu-row .mitsu-gap-sm .mitsu-stack-sm/md/lg .mitsu-grid .mitsu-grid-3
